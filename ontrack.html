What It Is

Membership, content, and app — one platform

OnTrack ties together three components into a single system: a WordPress plugin that handles all membership logic and access control, a WordPress theme that presents the same content on the web, and a branded React Native mobile app for members. The mobile app authenticates against your WordPress site and pulls navigation, content, and events from a REST API that enforces your membership and group rules on every response — so the web and the app are always in sync.

What OnTrack Does

Built for real community structure

Membership & Hierarchical Groups

Admin-assigned membership with groups that nest — a member in a child group automatically inherits access granted to its parent groups.

Audio & Video Library

Collections with a per-item access level — public, any logged-in visitor, or full members only — enforced consistently across web and app.

Events

Upcoming events with a banner image, date, time, location, and an optional live-stream link, surfaced on both the site and the mobile app.

Branded Mobile App

A tab bar driven entirely by your WordPress navigation menu, biometric sign-in, offline detection, and Universal Links for password reset that open the app directly.

Secure by Design

Dual-token JWT authentication with automatic refresh, and a self-updating plugin that checks for new versions without manual intervention.

Standing Up a New Community

Every community-specific value — name, branding, bundle IDs, deep links — lives in configuration, so a new instance doesn't require forking the codebase.

Who It's For

Organisations with a community to manage

Churches & Faith Groups

Membership records, group-based access, and a dedicated content library for sermons, teaching, and events.

Membership Associations

Demographic tracking, group rosters, and controlled access to member-only content and resources.

Content-Led Communities

Any organisation that wants a branded app and members area without building a platform from scratch.
